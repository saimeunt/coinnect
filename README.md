# Coinnect

## Run in dev

```
pnpm hardhat:node
pnpm hardhat:deploy:localhost
pnpm dev
```

## Test

`pnpm hardhat:test`

## Coverage

`pnpm hardhat:coverage`

## Deploy contracts

`pnpm hardhat:deploy:mumbai`

## Deploy dApp

`git push`

dApp is deployed on Vercel at:

https://coinnect.vercel.app/
